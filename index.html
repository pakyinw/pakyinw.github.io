bootdisk-keygen

Bootable Image for Private Key generator

This program generates a bootable bin file which can be mounted on USB Drive or virtual machine (qemu). It aims to help anyone who cannot afford the offline computer (no money, no space, etc.) for any cryptocurrency transaction.

You may say that you can mount any offline OS on the Virtual Machine but I think simple bootable USB should be safer.
